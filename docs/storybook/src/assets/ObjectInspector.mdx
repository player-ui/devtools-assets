# ObjectInspector Asset

> type: `object-inspector`

It provides an `ObjectInspector` component to display objects for debugging purposes.

## DSL Properties

| Property | Type | Required | Description |
| --- | --- | --- | --- |
| `binding` | `string` | false | The binding to the object to inspect. |
| `filter` | `boolean` | false | Flag to add the filter capability option to the object to inspect. |
