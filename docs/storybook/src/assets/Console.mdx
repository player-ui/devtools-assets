# Console Asset

> type: `console`
It provides a `Console` component that can be used to allow the user to interact with the Player-UI expression evaluator.

## DSL Properties

| Property | Type | Required | Description |
| --- | --- | --- | --- |
| `binding` | `string` | true | Binding to the history of evaluations. | 
| `exp` | `string` | true | Evaluation expression |
